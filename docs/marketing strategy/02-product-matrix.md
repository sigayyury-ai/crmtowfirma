## Подписка / Membership (мерчбокс + скидки)

Текущая подписка для участников комьюнити: мерчбокс + скидки и бонусы. Нужна как инструмент **retention** и “якорь” комьюнити между поездками.

| Метрика | Значение |
|---------|----------|
| Участников (сейчас) | 2 |
| Цена | €40 / мес |
| MRR | €80 / мес (~344 PLN) |
| ARR | €960 / год (~4,128 PLN) |
| Цель | Увеличить участников (связать с рефералкой и контентом) |

> Примечание: оценка для кемпов рассчитана как рабочая модель под план `492` участников при среднем размере группы `~14`: `492 / 14 ≈ 35` кемпов в год (в среднем `~3/мес`). В пиковые месяцы (`57` участников) потребуется либо 4-й кемп, либо увеличение размера группы/длительности/операционного режима — это нужно согласовать с операционным планом.
