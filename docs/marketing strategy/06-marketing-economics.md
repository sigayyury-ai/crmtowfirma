## LTV (Lifetime Value)

| Сегмент | LTV (PLN) | Среднее кол-во покупок |
|---------|-----------|------------------------|
| Новый участник | ~4,715 | 1 |
| Повторник | ~9,430 | ~2 |
| Core-комьюнити | ~14,145 | ~3 |
| **Средний LTV (оценка)** | ~6,255 | ~1.33 |

> Методика (оценка 2025): средний чек `~4,715` = `1,244,837 PLN / 264 Won`. `Repeat = 65` трактуем как дополнительные покупки к первым (то есть уникальных клиентов ≈ `264 - 65 = 199`). Тогда среднее число покупок на клиента ≈ `264 / 199 = 1.33`, а средний LTV ≈ `1,244,837 / 199 = 6,255 PLN`. Для `Повторник`/`Core` используем упрощённую модель 2 и 3 покупки соответственно — уточним, когда появится распределение покупок по клиентам.
