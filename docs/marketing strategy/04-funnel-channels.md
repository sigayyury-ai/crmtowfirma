# 4. Воронка и текущие каналы

> Источник: AARRR Funnel, Dave McClure

## Текущие каналы

> Важно: у нас два самых больших источника в 2024–2025 — **таргет в Instagram** и **органика из Instagram-аккаунта**.
> Это два разных сценария: часть людей приходит сразу по рекламе на сайт и оставляет заявку, часть “прогревается” контентом месяцами/годами и оставляет заявку позже.

| Канал | Доля трафика (%) | Доля продаж (%) | Как работает / примечания |
|-------|------------------|-----------------|
| Instagram — органика (аккаунт) | н/д | н/д | Долгий прогрев: подписка → сторис/посты → доверие → заявка через время |
| Instagram — таргет (реклама) | н/д | н/д | Быстрый путь: реклама → сайт/лендинг → заявка “сразу” |
| Сайт — органика (SEO/брендовый спрос) | н/д | н/д | Иногда находят через Google; SEO даёт ~200 органических кликов/мес (пока небольшой вклад). Переходы не из рекламы: поиск/прямые/закладки; часто коррелирует с “прогретыми” |
| Telegram | н/д | н/д | Доп. канал прогрева/коммуникации (уточнить вклад в лиды/сделки) |
| Рекомендации (referrals) | н/д | н/д | Есть реферальная программа и реферальные ссылки; работает, но пока не системно (нужно упаковать и регулярно запускать) |
| CPA/партнёрки (Telegram) | н/д | н/д | Срочный кейс: TG-группа для айтишников `devby.io` — даёт лидов регулярно (CPA-модель) |
| Микро‑инфлюенсеры | н/д | н/д | Коллаборации + CPA/реф‑коды: внешнее доверие и UGC‑контент |
| Амбассадоры (alumni) | н/д | н/д | Внутренний драйвер: офлайны, тёплые интро, рефералка и повторные покупки |
| Другое | н/д | н/д | Партнерства, упоминания, офлайн и т.д. |

## Каналы, которые не работают / не используем (2024–2025)

- **Google Ads** — не используем: нет набора ключей/семантики под спрос (требует отдельной работы по ключам и лендингам).
- **TikTok Ads** — были попытки запусков, но не давало заметных результатов.
- **LinkedIn Ads** — слишком дорого для наших бюджетов (низкая эффективность при текущем CAC).

## План 2026 по воронке (рабочая версия)

Цель — покрыть план `670` участников (кемпы/коливинги/викенды) при сохранении экономики и усилении органики/рефералок.

| Метрика | Цель 2026 | Основание |
|--------|-----------|----------|
| Won (покупки/участники) | 670 | план продукта 2026 |
| Conversion MQL → Won | 40% | ориентир ≈ 2025 (39.1%) |
| MQL (лиды) | ~1,675 | `670 / 0.40` |
| SQL / сделки в Pipedrive (лиды) | 1,138 | план “сделок/лидов” на 2026 (Pipedrive) |

> Важно: чтобы этот план сходился, нужно фиксировать термины (MQL / SQL / Deal created / Won) и атрибуцию каналов в Pipedrive (UTM + `First touch`).
